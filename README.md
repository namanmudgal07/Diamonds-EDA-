## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Diamonds dataset to extract meaningful insights and understand relationships between various features affecting diamond prices.

The analysis includes data cleaning, statistical exploration, and visualization techniques to identify trends, correlations, and anomalies.

---

## 🎯 Objectives

- To explore the structure and characteristics of the dataset  
- To perform data cleaning and preprocessing  
- To analyze relationships between variables  
- To visualize patterns using graphs and charts  
- To derive meaningful insights from the data  

---

## 📊 Dataset Description

The dataset consists of various attributes of diamonds:

| Feature   | Description |
|----------|------------|
| price    | Price of the diamond |
| carat    | Weight of the diamond |
| cut      | Quality of the cut (Fair, Good, Very Good, Premium, Ideal) |
| color    | Diamond color grading |
| clarity  | Clarity level |
| depth    | Total depth percentage |
| table    | Width of top of diamond |
| x, y, z  | Dimensions (length, width, depth in mm) |

## 🔍 Steps Performed

### 1. Data Loading
- Imported dataset using Pandas  
- Checked dataset shape and structure  

### 2. Data Cleaning
- Checked for missing values  
- Removed or handled inconsistencies  
- Verified data types  

### 3. Statistical Analysis
- Generated summary statistics  
- Identified distributions and spread  

### 4. Data Visualization
- Histograms for distribution analysis  
- Box plots for outlier detection  
- Scatter plots for relationships  
- Heatmap for correlation analysis  

### 5. Feature Analysis
- Studied impact of features like **carat, cut, clarity** on price  
- Identified strong correlations  

---

## 📈 Key Insights

- 💰 **Price is highly correlated with carat weight**  
- 💎 Better **cut quality** tends to increase price  
- 📊 Certain features contain **outliers**  
- 🔗 Strong relationships exist between dimensions (x, y, z) and price
